# Phase 8 — UI Design Contract

> Visual and interaction contract for the Panel & Tab Restructure phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (native macOS app) |
| Preset | not applicable |
| Component library | SwiftUI + AppKit (NSPanel) |
| Icon library | SF Symbols (Apple system icons) |
| Font | San Francisco (macOS system font via SwiftUI `.font()` modifiers) |

**Note:** This is a native macOS menu bar application. No web component libraries, CSS frameworks, or design tokens apply. All visual styling uses SwiftUI's built-in semantic modifiers (`.font()`, `.foregroundStyle()`, `.padding()`), which automatically adapt to macOS appearance settings (light/dark mode, accessibility sizes, accent color).

---

## Spacing Scale

Declared values (must be multiples of 4). Derived from existing codebase patterns established in v1.0:

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-text gaps (AggregateHeaderView arrow-to-speed), VStack spacing in stat cards |
| sm | 8px | Compact element spacing: InterfaceRowView icon-to-name gap, stat card internal padding, interface list top inset, interface row vertical padding, CumulativeStatsView card gap, divider vertical padding in DashboardView, corner radius on stat cards |
| md | 16px | Default horizontal padding on all content sections, segmented control horizontal padding, popover header top/bottom padding, divider horizontal inset, chart horizontal padding, stat cards horizontal padding |
| lg | 24px | AggregateHeaderView bottom padding (visual weight under primary speed readout) |
| xl | 32px | Not used in this phase |
| 2xl | 48px | Not used in this phase |
| 3xl | 64px | Not used in this phase |

Exceptions:
- 6px -- tooltip internal padding and tooltip indicator-to-text spacing (HistoryChartView tooltip, inherited from v1.0)
- 12px -- speed value pair spacing in InterfaceRowView (upload-to-download gap), chart top padding below time range picker
- 44px -- minimum row height for InterfaceRowView (accessibility touch target, inherited from v1.0)

## Typography

All sizes use the macOS San Francisco system font. SwiftUI semantic font styles are used rather than raw pixel values for automatic Dynamic Type support.

| Role | SwiftUI Style | Approximate Size | Weight | Line Height | Usage |
|------|--------------|------------------|--------|-------------|-------|
| Display | `.title2.monospacedDigit()` | 22pt | `.semibold` (600) | System default (~1.2) | AggregateHeaderView: primary upload/download speed values |
| Heading | `.title3` | 20pt | `.semibold` (600) | System default (~1.2) | StatCardView: primary cumulative total (e.g., "1.24 GB") |
| Body | `.body` / `.body.monospacedDigit()` | 13pt | `.regular` (400) | System default (~1.4) | InterfaceRowView: interface names and per-interface speed values |
| Label | `.caption` | 10pt | `.regular` (400) | System default (~1.3) | StatCardView title labels ("Today", "This Week", "This Month") |
| Small | `.caption2` | 10pt | `.regular` (400) | System default (~1.3) | StatCardView per-direction breakdowns, chart tooltip text |
| Secondary | `.subheadline` / `.callout` | 12pt / 12pt | `.regular` (400) | System default (~1.4) | Empty state messages |

**Weights used (exactly 2):** `.regular` (400) for body text, labels, and secondary content; `.semibold` (600) for primary speed values and stat totals.

**Monospaced digits:** All numeric speed/byte values use `.monospacedDigit()` to prevent layout jitter during live updates.

## Color

This application uses macOS semantic colors exclusively. All colors adapt automatically to light mode, dark mode, and user-configured accent colors. No hardcoded hex values.

| Role | SwiftUI Value | Usage |
|------|--------------|-------|
| Dominant (60%) | System window background (automatic via NSPanel) | Panel background, view backgrounds -- provided by the macOS window chrome and SwiftUI default backgrounds |
| Secondary (30%) | `.quaternary.opacity(0.5)` in `RoundedRectangle(cornerRadius: 8)` | Stat card backgrounds (CumulativeStatsView); `.ultraThinMaterial` for chart tooltip background |
| Accent (10%) | `Color.accentColor` | Reserved for: (1) upload/download arrow indicators in AggregateHeaderView, (2) chart download bar fill color. Nothing else. |
| Secondary text | `.secondary` | Interface row icons, stat card labels, stat card per-direction breakdowns, empty state text, chart upload bar fill (`.secondary.opacity(0.7)`) |
| Primary text | `.primary` | All primary content text: speed values, interface names, stat totals |
| Divider | SwiftUI `Divider()` | Horizontal separators between aggregate header and interface list, and between live speeds section and history section (D-05) |

**Accent reserved for:** AggregateHeaderView directional arrows (up/down), HistoryChartView download bar series color. Never used for backgrounds, borders, or general interactive elements.

**Chart color scale:**
- Download bars: `Color.accentColor`
- Upload bars: `Color.secondary.opacity(0.7)`

### Dashboard View Layout (D-04, D-05, D-06)

Top-to-bottom reading order within a single `ScrollView`:

```
+----------------------------------------------+
|          [ Dashboard ] [ Preferences ]        |  <- Segmented control (16px padding all sides)
+----------------------------------------------+
|                                               |
|   arrow.up  12.4 MB/s      arrow.down 45.2 MB/s |  <- AggregateHeaderView (16px h-pad, 24px bottom-pad)
|                                               |
|   ----------------------------------------    |  <- Divider (16px horizontal inset)
|                                               |
|   [wifi]  Wi-Fi           up 1.2   down 3.4   |  <- InterfaceRowView (8px top-pad, 4px between rows)
|   [cable] Ethernet        up 11.2  down 41.8  |     44px min height per row
|                                               |
|   ----------------------------------------    |  <- Divider (D-05: 16px h-inset, 8px vertical pad)
|                                               |
|   [ 1H ] [ 24H ] [ 7D ] [ 30D ]              |  <- Time range picker (16px h-pad, 8px top-pad)
|                                               |
|   +--------------------------------------+    |  <- HistoryChartView (16px h-pad, 12px top-pad)
|   |   [chart bars]                       |    |     200px fixed height
|   +--------------------------------------+    |
|                                               |
|   +--------+ +--------+ +--------+           |  <- CumulativeStatsView (16px h-pad, 16px bottom-pad)
|   | Today  | | Week   | | Month  |           |     8px gap between cards, 8px internal padding
|   | 1.24GB | | 8.7 GB | | 34 GB  |           |
|   | d  u   | | d  u   | | d  u   |           |
|   +--------+ +--------+ +--------+           |
+----------------------------------------------+
```

**Scroll behavior (D-06):** Single outer `ScrollView` wraps all content. No nested ScrollViews. The MetricsView's internal `ScrollView` around the interface list MUST be removed when composed into DashboardView. The aggregate header scrolls off-screen if user scrolls down.

### Context Menu (D-08)

Menu items in order:
1. "Dashboard" -- opens panel to Dashboard tab
2. "Preferences" -- opens panel to Preferences tab
3. Separator
4. "About Bandwidth Monitor" -- standard about panel
5. "Quit Bandwidth Monitor" (key equivalent: Cmd+Q) -- terminates app

## Interaction Contract

| Interaction | Behavior | Source |
|-------------|----------|--------|
| Left-click status item | Toggle panel visibility (show centered / close) | D-09 |
| Right-click status item | Open context menu | Existing pattern |
| Click outside panel | Panel closes (resignKey) | D-02 |
| Switch to another app | Panel closes (resignKey) | D-02 |
| Click "Dashboard" in context menu | Open panel, select Dashboard tab | D-08, D-09 |
| Click "Preferences" in context menu | Open panel, select Preferences tab | D-08 |
| Segmented control tap | Switch between Dashboard and Preferences | Existing pattern |
| Scroll in Dashboard | Single continuous scroll through live speeds and history | D-06 |

**Transitions:**
- Panel appear: Instant (no animation, per D-03)
- Panel disappear: Instant (no animation, per D-03)
