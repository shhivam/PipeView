    **Add query methods to AppDatabase** (extension, not modifying existing code).

    **`fetchChartData(tier:since:)`** (per D-07, POP-02):
    ```swift
    func fetchChartData(tier: AggregationTier, since: Date) throws -> [ChartDataPoint]
    ```
    - Uses raw SQL: `SELECT bucketTimestamp, SUM(totalBytesIn) AS totalIn, SUM(totalBytesOut) AS totalOut FROM {tier.tableName} WHERE bucketTimestamp >= ? GROUP BY bucketTimestamp ORDER BY bucketTimestamp ASC`
    - Argument: `since.timeIntervalSince1970`
    - Maps rows to ChartDataPoint with `Date(timeIntervalSince1970: row["bucketTimestamp"])`
    - Uses `dbWriter.read { }` (synchronous read, fine for small aggregated tables per research Open Question 3)

    **`fetchCumulativeStats(since:)`** (per D-09, D-11, POP-04):
    ```swift
    func fetchCumulativeStats(since: Date) throws -> (totalIn: Double, totalOut: Double)
    ```
    - Queries hour_samples for "Today" accuracy (per research recommendation -- hour_samples is more current than day_samples for partial days)
    - SQL: `SELECT COALESCE(SUM(totalBytesIn), 0) AS totalIn, COALESCE(SUM(totalBytesOut), 0) AS totalOut FROM hour_samples WHERE bucketTimestamp >= ?`
    - Note: All three stats cards (Today, This Week, This Month) call this with different `since` dates. The caller computes the local timezone start-of-period and passes it (per Pitfall 6: UTC vs local time).
