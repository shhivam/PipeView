    **Modify `StatusBarController.swift`** (per D-15, Pitfall 4):
    - Remove the hardcoded `private let displayMode: DisplayMode = .auto` and `private let unitMode: SpeedFormatter.UnitMode = .auto`.
    - In `startObserving()`, replace `self.displayMode` and `self.unitMode` with live reads from UserDefaults:
      ```swift
      let displayModePref = DisplayModePref(
          rawValue: UserDefaults.standard.string(forKey: PreferenceKey.displayMode) ?? "auto"
      ) ?? .auto
      let unitModePref = UnitModePref(
          rawValue: UserDefaults.standard.string(forKey: PreferenceKey.unitMode) ?? "auto"
      ) ?? .auto

      let text = self.textBuilder.build(
          speed: speed,
          mode: displayModePref.toDisplayMode(),
          unit: unitModePref.toUnitMode(),
          hasInterfaces: hasInterfaces
      )
      ```
    - Per research Pattern 5 simpler alternative: since startObserving re-registers via withObservationTracking on every NetworkMonitor change (~2 seconds), it will naturally pick up preference changes within one poll cycle. No explicit KVO or NotificationCenter observation needed.

    **Modify `AppDelegate.swift`** (per Pitfall 7):
    - Add a UserDefaults observer for the update interval preference. After `networkMonitor.start()`, add:
      ```swift
      // Observe update interval preference changes
      NotificationCenter.default.addObserver(
          forName: UserDefaults.didChangeNotification,
          object: nil,
          queue: .main
      ) { [weak self] _ in
          guard let self else { return }
          let intervalPref = UpdateIntervalPref(
              rawValue: UserDefaults.standard.integer(forKey: PreferenceKey.updateInterval)
          ) ?? .twoSeconds
          self.networkMonitor.pollingInterval = intervalPref.duration
      }
      ```
    - Note: UserDefaults.didChangeNotification fires on ANY UserDefaults change but the overhead is negligible (one if check, runs rarely). This is simpler than per-key KVO.
    - Also set the initial polling interval from UserDefaults before `networkMonitor.start()`:
      ```swift
      let initialInterval = UpdateIntervalPref(
          rawValue: UserDefaults.standard.integer(forKey: PreferenceKey.updateInterval)
      ) ?? .twoSeconds
      networkMonitor.pollingInterval = initialInterval.duration
      ```
